# Kiraak AI

**AI-Powered Personal Investment Platform | Target: 2% Weekly Returns**

---

## Problem Statement

95% of retail investors lose money due to lack of institutional-grade AI analysis. Kiraak AI democratizes hedge fund-level intelligence, providing real-time stock recommendations, risk scoring, and portfolio management across Indian and US markets.

---

## Users & Context

**Target Users:**
- Retail investors in India (50M+) and US (100M+) seeking consistent returns
- Swing traders, momentum traders, event-driven traders
- Portfolio managers needing AI-powered stock screening

**Context:**
- Real-time market data from Yahoo Finance & Alpha Vantage APIs
- Multi-sector analysis (Technology, Healthcare, Finance, Energy, Semiconductors)
- Supports both Indian (NSE/BSE) and US (NYSE/NASDAQ) markets

---

## Solution Overview

Kiraak AI combines three core AI engines:

1. **Momentum Scoring Engine**: Calculates price + volume momentum (0-1 scale) to identify trending stocks
2. **Risk Scoring Algorithm**: 9-factor model (Beta, Sharpe Ratio, Volatility, Financial Health) outputs 0-100 risk score
3. **AI Recommendation Engine**: Multi-factor decision tree generates BUY/SELL/HOLD with confidence scores (0-10)

**Key Features:**
- 2% Weekly Returns Filter: Identifies stocks with 2%+ expected returns
- Sector Analysis: Real-time momentum scoring across sectors
- Portfolio Management: Multi-portfolio tracking with P/L analytics
- Trading Recommendations: Entry zones, price targets, stop-loss levels

**Architecture:**
```
User → Next.js Frontend → API Routes → AI Engines → Market Data APIs → SQLite DB
```

## Setup & Run

### Prerequisites
- Node.js 18+
- npm 9+
- SQLite3

### Installation Steps

1. **Clone & Install**
   ```bash
   git clone <repo-url>
   cd Kiraak
   npm install
   ```

2. **Environment Setup**
   ```bash
   cp .env.example .env
   # Add: JWT_SECRET, SESSION_SECRET, API_KEY_ALPHA_VANTAGE (optional)
   ```

3. **Database Migration**
   ```bash
   npm run db:migrate
   npm run db:sync
   ```

4. **Run Development Server**
   ```bash
   npm run dev
   # Open http://localhost:3000
   ```

5. **Production Build**
   ```bash
   npm run build
   npm start
   ```

## Models & Data

### AI Models Used

1. **Momentum Scoring Algorithm**
   - **Input**: Historical prices, volume data
   - **Output**: Momentum score (0-1)
   - **Logic**: Weighted average of price momentum + volume momentum

2. **Risk Scoring Algorithm**
   - **Input**: Beta, Sharpe Ratio, Sortino Ratio, Implied Volatility, Current Ratio, Short Interest, Max Downside Risk
   - **Output**: Risk score (0-100), Risk category (LOW/MEDIUM/HIGH)
   - **Logic**: Base score 50, adjusted by 9 factors

3. **AI Recommendation Engine**
   - **Input**: Momentum score, price change %, volume
   - **Output**: BUY/SELL/HOLD recommendation
   - **Logic**: Decision tree with weighted scoring

### Data Sources

- **Yahoo Finance API** (Primary): Free, real-time market data, supports IN/US markets
- **Alpha Vantage API** (Fallback): Requires API key, detailed historical data
- **Proprietary Stock Analysis**: JSON files with fundamental/technical data (Data_*.json, Data2_*.json)

### Data Licenses

- Yahoo Finance: Public API, no license required
- Alpha Vantage: Free tier (5 calls/min), paid tiers available
- Stock analysis data: Proprietary, generated by AI analysis

## Evaluation & Guardrails

### Hallucination Mitigations

- **Multi-source validation**: Cross-reference Yahoo Finance + Alpha Vantage
- **Confidence scores**: All recommendations include 0-10 confidence level
- **Risk scoring**: High-risk stocks flagged automatically
- **Stop-loss enforcement**: Trading recommendations include mandatory stop-loss levels

### Bias Mitigations

- **Sector diversification**: Recommendations span multiple sectors
- **Market-neutral approach**: Equal weight to Indian and US markets
- **Risk-adjusted returns**: Sharpe/Sortino ratios prevent high-risk bias
- **Transparent algorithms**: All scoring logic is deterministic and auditable

### Evaluation Metrics

- **Recommendation Accuracy**: Track BUY recommendations vs actual returns
- **Risk Score Accuracy**: Compare predicted risk vs actual volatility
- **Weekly Returns Target**: Monitor % of users achieving 2%+ weekly returns
- **User Retention**: Track active users, portfolio performance

## Known Limitations & Risks

### Limitations

- **Real-time data delays**: API rate limits (200ms between requests)
- **Historical data**: Limited to available API data (no proprietary historical database)
- **Market coverage**: Focus on liquid stocks, may miss small-cap opportunities
- **AI accuracy**: Recommendations based on technical analysis, not fundamental research

### Risks

- **Market volatility**: AI accuracy drops in extreme market conditions
- **Regulatory**: Financial advice regulations vary by jurisdiction (mitigated: educational tool with disclaimers)
- **Data quality**: API outages or incorrect data affect recommendations
- **Over-reliance**: Users may ignore risk warnings (mitigated: mandatory risk scoring display)
